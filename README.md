

# рҹҸҘ MediBot - Medical AI Assistant

Specialized in **Hypertension & Kidney Disease in Ghana** using **RAG (Retrieval-Augmented Generation)**.

## рҹҺҜ Features

- рҹ—ЈпёҸ **AI Chat** powered by OpenRouter LLM
- рҹ“ҡ **Ghana Health Service** document integration
- рҹ—ғпёҸ **Pinecone** vector database for semantic search
- рҹ“Ө **PDF Upload** for new medical documents
- рҹ‘Ө **User Authentication** (Login/Signup)
- вӯҗ **Feedback System** for continuous improvement
- рҹӣЎпёҸ **Rate Limiting** for security
- рҹ“Ҡ **Admin Dashboard** for statistics
